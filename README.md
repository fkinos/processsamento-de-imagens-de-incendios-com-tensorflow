# processsamento de imagens de incendios florestais com Tensorflow
O objetivo desse projeto de faculdade foi desenvolver um programa que utilize de processamento de imagens para reconhecer imagens de incendios florestais via satélite. 

Data Set utilizado para o treinamento:
https://www.kaggle.com/datasets/abdelghaniaaba/wildfire-prediction-dataset

## Como utilizar:

Execute:
``` 
python predict.py <file_path>
```

## Dependências
* Pillow
* Tensorflow
